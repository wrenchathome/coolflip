REM This is a cool BadUSB script for Windows

REM First we open windows notepad
DELAY 1000
GUI r
DELAY 500
STRING notepad
DELAY 500
ENTER
DELAY 750

REM Now lets type something cool
STRING something cool

REM Lets wait a bit to give you time to appreciate this line
DELAY 1000

ENTER
REM how about we type something else now?
STRING something else


REM adding a bit more of a delay, aka waiting time :)
DELAY 1000


REM Now that we typed cool stuff, lets close the notepad like nothing ever happened!
ALT F4
DELAY 200
TAB
DELAY 50
ENTER
